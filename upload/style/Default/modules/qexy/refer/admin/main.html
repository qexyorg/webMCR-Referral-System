Название
- Название модуля реферальной системы (Отображается в заголовке страницы и хлебных крошках)
Рефералов на страницу
- Кол-во выводимых рефералов на одну страницу
Подозрительных пользователей на страницу
- Кол-во выводимых подозрительных пользователей на одну страницу
Уровень доступа к реферальной системе
- Минимальный уровень доступа к реферальной системе
Уровень доступа к панели управления
- Минимальный уровень доступа к панели управления статическими страницами
Награда
- Сумма, которую получит пригласивший игрок на свой баланс
Использовать MailBox
- Совместимость с модулем MailBox, которая добавляет дополнительные кнопки, для отправки личных сообщений
Использовать систему пользователей
- Совместимость с модулем User System, которая подсвечивает выводимых пользователей, ссылкой на их профиль
