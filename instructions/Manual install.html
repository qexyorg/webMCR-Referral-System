Ручная установка

Внимание! Ручную установку стоит использовать только в том случае, если файл instruments/menu_items.php был изменен

В противном случае, воспользуйтесь автоматической установкой

Открываем instruments/menu_items.php и перед строчкой

'exit' => array (

Добавляем

'qx_refer' => array (

'name' => 'Реферальная система',
'url' => Rewrite::GetURL('refer'),
'parent_id' => -1,
'lvl' => 2000,
'permission' => -1,
'active' => false,
'inner_html' => '',
),
  • Заливаем все файлы, кроме instruments/menu_items.php, в главный каталок webMCR
  • Переходим по адресу http://mysite.com/go/refer
  • После того как появится сообщение об успешной установке, удаляем папку rs_install
  • Готово

Ссылку на реферальную систему, т.е. http://mysite.com/go/refer можно поместить в любое место сайта.

Например, для минипрофиля, можно сделать так: открываем файл style/Default/mini_profile.html и перед

<p><a href="login.php?out=1" style="width: 128px;" class="btn btn-small" >Выход</a></p>

Добавляем

<p><i class="icon-bullhorn"></i> <b>Другое:</b> <a href="<?=BASE_URL?>go/refer" > Рефералы </a></p>
